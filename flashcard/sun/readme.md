
## Project Design

name: 
ACAP - A character (concept) A Page  
一字一页


### Flashcard page 

each page explains a concept
 
- consists of 20 blocks arranged in 5 rows (1-5) by 4 columns (A-D)
	identified by their coordinates: 1A, 1B, 1C, 1D, 2A, 2B， ..., 5C， 5D。
- displays 
	- pronounciation (sound)
	- shape
	- images 
	- meaning 
	- usage
	- idiom, phrases
	- sentences
	- paragraph
	- story
	- poem
	
- other resources
	- BaiDu 百科
	- Wikipedia
	
- QR code 
	- URL link to website
